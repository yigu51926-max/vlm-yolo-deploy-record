## 四、大小模型协同策略设计

编写大小模型协同策略文档：

docs/model_collaboration_strategy.md

文档明确了 YOLO26 与 Qwen3-VL 的分工：

YOLO26 作为小模型，负责多路 RTSP 视频流的高频目标检测；
Qwen3-VL 作为大模型，负责对 YOLO 触发的关键帧进行图像语义理解和风险分析；
系统不让 Qwen3-VL 处理每一帧，而是通过 YOLO26 筛选关键帧，再交给 Qwen3-VL 分析，从而兼顾实时性和语义理解能力。

当前设计采用的最小触发规则为：当 YOLO26 检测到 person，且置信度大于等于 0.60 时，保存当前帧并触发 Qwen3-VL 分析。
